Ejercicio 2

Crea un script que compruebe que los datos de un formulario están escritos correctamente, los campos del formulario son: Nombre completo, número de teléfono y dirección. El nombre completo solo puede tener caracteres, el número de teléfono solamente admite dígitos, y la dirección puede tener caracteres alfa-numéricos. Debes comprobar que los valores introducidos cumplen con el formato. Para extraer los elementos debes usar getElementByTagName. Si algún elemento no dtuviera el formato adecuado debes borrar el campo y mostrar un mensaje de error.(1'5 punto)
