Realiza los siguientes ejercicios utilizando JavaScript

1 - Modifica el ejemplo ajax3 para utilizar la función fetch() en vez de HttpRequest. Muestra los mismo valores por pantalla

2 - Cagar el archivo catalogo.xml, y muestra el titulo del CD, el artista y el año de todos aquellos CDs que salieron a partir del año 1990.

3 - Modifica el ejercicio anterior para que cada elemento tenga una opción de borrar, guarda esta información el almacenamiento local con los elementos que no se han borrado. Añade una opción para volver a cargar los datos del fichero y otra para borrar todo el almacenamiento local.

4 - Crea un script que almacene un cronometro en memoria local, debe tener las siguientes opciones, iniciar/reanudar, parar y borrar.

5 - Utilizando el fichero supers.json, muestra aquellos superhéroes que tenga 3 o menos superpoderes.

6 - Crea una pagina que tenga varios estilos, y que con algún elementos puedas cambiar dichos estilos, el tema o estilo se debe guardar en memoria local y cuando se inicie la pagina tiene que leer dicha información y mostrar la pagina con la configuración seleccionada.
